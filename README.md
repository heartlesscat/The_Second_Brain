# 第二大脑

这是一个可直接用 Obsidian 打开的本地 Vault，用来承载你和 Codex 共同维护的个人第二大脑。

## 目录概览

- `CODEX.md`：AI 总指南与长期规则
- `任务清单.md`：唯一任务入口
- `对话历史.md`：会话级存档
- `记忆库/`：情景记忆、语义记忆、强制规则
- `任务归档/`：已完成任务归档
- `01-内容/`：文章与资料
- `02-笔记/`：方法、卡片、主题笔记
- `03-项目/`：项目工作台
- `04-工具/`：工具与自动化说明

## 打开方式

1. 启动 Obsidian
2. 选择“Open folder as vault”
3. 打开 `第二大脑`

## 推荐下一步

- 补一个个人主页笔记
- 挑选是否启用 Dataview、Templater、Calendar
- 把你当前最关心的项目放进 `03-项目/`
