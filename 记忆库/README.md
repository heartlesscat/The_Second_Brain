# 记忆库说明

记忆库是这套系统的核心，按三层结构组织：

## 1. 情景记忆

- 保存具体事件、会话和决策
- 适合记录一次任务、一次讨论、一次复盘
- 推荐模板：
  - 背景
  - 发生了什么
  - 做了哪些决定
  - 下一步是什么

## 2. 语义记忆

- 从情景记忆中提炼可复用的方法、模式和认知
- 适合长期维护，例如：
  - 你的工作偏好
  - 常见项目套路
  - 稳定有效的决策准则

## 3. 强制规则

- 存放需要长期遵循的系统规则
- 只有确认足够稳定、反复有效时才写入

## 推荐流转路径

`对话/事件 -> 情景记忆 -> 语义记忆 -> 强制规则`

## 日常使用建议

1. 新任务先看 `任务清单.md`
2. 重要事件结束后补一条情景记忆
3. 每周回顾情景记忆，提炼到语义记忆
4. 只有真正稳定的约束才进入强制规则
