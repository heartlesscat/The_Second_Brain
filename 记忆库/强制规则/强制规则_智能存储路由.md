# 强制规则：智能存储路由

每次创建文件时默认执行以下步骤：

1. 判断内容类型
2. 查询存储规则
3. 构建目标路径
4. 验证路径是否仍在 Vault 内
5. 创建或更新文件
6. 告知用户保存位置和理由

## 存储规则

- 文章、资料整理：`01-内容/`
- 主题笔记、方法卡片：`02-笔记/`
- 项目资料、项目计划：`03-项目/`
- 工具脚本、自动化说明：`04-工具/`
- 系统规则、记忆与元数据：根目录或 `记忆库/`

## 边界

- 默认不向 Vault 外写文件
- 允许读取 Vault 外文件用于分析
- 如果用户指定外部保存位置，先提醒风险，再建议放回 Vault 内部
